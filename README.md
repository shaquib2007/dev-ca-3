

A simple template repository with JavaScript code for variable manipulation.

## Description

This repository consists of the following files:

- [`variable.js`](https://github.com/shaquib2007/dev-ca-3/blob/main/variable.js): Contains JavaScript code that demonstrates variable usage and operations.

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/shaquib2007/dev-ca-3.git
   ```

2. Open the `variable.js` file to review the example code or run it with Node.js:
   ```sh
   node variable.js
   ```
